# dnsmgr
## Описание

Клиент-серверное gRPC приложение для управления DNS-серверами.

Приложение позволяет управлять DNS-серверами `/etc/resolv.conf` на удаленным компьютере.

## Использование
1. На компьютере с запущенным серверным приложением, можно выполнять следующие функции:
- Добавить DNS-сервер
- Удалить DNS-сервер
- Получить список всех DNS-серверов

2. CLI клиент позволяет выполнять функции на удалённом компьютере. Используйте `-h` или `-help` для справки.
